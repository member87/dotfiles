# Dotfiles

Various configuration files for my ArchLinux setup.

Using [YADM](https://yadm.io/) to manage dotfiles

## Window manager setup:
- i3wm (sometimes use awesomewm)
- polybar
- i3 resurect to restore i3 programs and layout
- alacritty
- rofi
- picom
